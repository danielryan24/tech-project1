Code & Design

HTML

HTML adalah bahasa markup standar untuk membuat halaman website. Bahasa ini berfungsi untuk membuat struktur website hingga menyusun format teks dan gambar pada halaman web.

CSS

CSS adalah singkatan dari cascading style sheets, yaitu bahasa yang digunakan untuk menentukan tampilan dan format halaman website. Dengan CSS, Anda bisa mengatur jenis font, warna tulisan, dan latar belakang halaman.

Figma

Figma adalah aplikasi desain grafis yang berfungsi untuk membuat desain tampilan sebuah website atau aplikasi. Figma juga memudahkan para pengembangnya untuk selalu meng-update secara berkala terkait fitur-fitur yang dibutuhkan.

Office

Ms.Word

Microsoft Word merupakan program aplikasi pengolah kata (word processor) yang yang biasa digunakan untuk membuat laporan, membuat dokumen berbentuk surat kabar, membuat label surat, membuat table pada dokumen, dan masih banyak lagi dukumen- dokumen lain yang biasa dibuat dengan menggunakan Microsoft Word.

Ms.Excel

Microsoft Excel adalah aplikasi yang memungkinkan input data berupa tulisan, teks, angka, atau fungsi kompleks lainnya. Aplikasi ini juga dikenal dengan sebutan spreadsheet (lembar kerja elektronik).

Ms.Powerpoint

Microsoft Powerpoint adalah sebuah program aplikasi microsoft office yang berguna sebagai media presentasi dengan menggunakan beberapa slide. Kehadiran powerpoint membuat sebuah presentasi berjalan lebih mudah dengan dukungan fitur yang sangat menarik dan canggih.
